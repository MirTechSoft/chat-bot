# 💬 Chat-Bot App

A simple and intelligent chat bot built using **PHP** and **MySQL**.  
This bot handles user messages, stores chat history in a database, and provides predefined smart replies.

---

## 🚀 Features

- ✅ Real-time user message handling
- ✅ Smart auto-replies (50+ common responses)
- ✅ Clean and responsive chat interface
- ✅ Messages stored in MySQL database
- ✅ PHP-powered backend logic
- ✅ Easy to deploy on localhost or live server

---

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
